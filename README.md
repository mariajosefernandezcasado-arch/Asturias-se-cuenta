El alumnado investigará manifestaciones culturales, naturales y patrimoniales de Asturias, elaborando productos comunicativos multimodales dirigidos a una audiencia real, favoreciendo así el uso funcional de la lengua en contextos significativos.
La propuesta culmina con dos productos finales:
•	Una infografía interactiva digital.
•	Un podcast divulgativo. 
Ambos productos integran comprensión lectora, producción escrita, expresión oral, alfabetización mediática y competencia digital.
